Welcome to the **PHP Fundamentals** repository!

This project demonstrates basic PHP topics including:

- Arithmetic Operators  
- Arrays   
- Conditional Statements  
- Loops  

Each folder contains small scripts or examples related to a fundamental concept in PHP.

## 🧠 Topics Covered

### 🔹 Arithmetic Operators
Learn how to perform basic computations:
- `+`, `-`, `*`, `/`, `%`

### 🔹 Arrays
Work with:
- Indexed arrays  
- Associative arrays  
- Multidimensional arrays

### 🔹 Conditional Statements
Control program flow with:
- `if`  
- `else`  
- `elseif`  
- `switch`

### 🔹 Loops
Repeat actions with:
- `for`  
- `while`  
- `do…while`  
- `foreach`
